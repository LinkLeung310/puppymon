# Puppymon

Puppymon is a mobile-first web prototype for catching real-world dogs as collectible game cards.

## Features

- Phone camera capture flow with `capture="environment"`
- Playful catch animation and collection loop
- Procedural puppy cards with name, rarity, level, and stats
- Nearby-finds map mock
- Local collection persistence with `localStorage`

## Run

Open `index.html` in a browser. For live camera access, serve it from HTTPS or localhost.
